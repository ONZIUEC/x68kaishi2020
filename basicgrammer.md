### ソースコード

```C
#include <stdio.h>

int main() {
    int A[5] = {6, 8, 0, 0, 0};
    int sum = 0;
    for(int index = 0; index < 5; index++) {
        sum += A[index];
    }
    printf("sum = %d\n", sum);
    return 0;
}
```
